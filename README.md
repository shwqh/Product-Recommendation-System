# Product Recommendation System Using Online Retail Data

## Project Overview
This project presents a Product Recommendation System developed using the Online Retail II dataset. The objective is to analyze customer purchasing behavior and generate product recommendations using different recommendation techniques.

## Dataset
The project uses the Online Retail II dataset, which contains transactional records from an online retail store.

Dataset Source:  
https://archive.ics.uci.edu/dataset/502/online+retail+ii

## Project Objectives
- Analyze customer purchase behavior.
- Identify popular products.
- Discover product associations using Market Basket Analysis.
- Generate personalized recommendations using Collaborative Filtering.
- Compare different recommendation approaches.

## Recommendation Approaches

### 1. Popularity-Based Recommendation
Recommends the most frequently purchased products to all users.

### 2. Association Rules (Apriori)
Uses Market Basket Analysis to discover relationships between products frequently purchased together.

### 3. Collaborative Filtering
Uses customer similarity based on purchase history to generate personalized recommendations.

## Results
The project demonstrates that different recommendation techniques provide different benefits:
- Popularity-Based Recommendation is simple and easy to implement.
- Association Rules help discover product relationships.
- Collaborative Filtering provides more personalized recommendations.
